## Migrasi ke Subdomain Baru

Ketika subdomain baru sudah siap dengan data lengkap:
1. Buka Filament Admin → Perpustakaan → Sumber Jurnal
2. Edit setiap jurnal dan ubah Base URL dan Feed URL
3. Ganti feed_type dari 'atom' ke 'oai' jika menggunakan OAI-PMH
4. Jalankan Sync untuk mengambil artikel baru
